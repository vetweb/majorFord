Принцип работы Space Grid ?

Робот открывает ордера на покупку и на продажу, используя индикаторы и основываясь на общем движении импульса цены. Создает сетку из ордеров. При определенном проценте увеличения просадки, автоматически переходит на таймфрейм H4, D1 и обратно, при снижении процента просадки.

Робот использует 3 скальпинговых стратегии для определения точек входа, что увеличивает эффективность торговли. Выставленные ордера закрываются намного быстрее и их значительно больше, чем при одновалютной системе, что обеспечивает плавный рост депозита при небольших просадках.

Мультивалютная торговля используется с индивидуальными настройками по каждой паре, что обеспечивает диверсификацию торгов и уменьшение рисков. То есть по каждой паре торгуем значительно уменьшенными лотами по сравнению с одновалютной стратегией. Одновременно больших просадок по некоторым валютным парам не будет.

Торгует робот на таких валютных парах как: eurchf, eurjpy, eurusd, usdcad, usdchf,usdjpy.
